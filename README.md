# Vue Svg Icons

Create and use your own icons, or use only your favorites in your vue.js project.

## Install

```bash
npm i --save-dev vue-svg-icons
# or
yarn -D vue-svg-icons
```

## Options

| Property | Type                 | Default        |
| -------- | -------------------- | -------------- |
| path     | `String`             | `''`           |
| title    | `String`             | `''`           |
| color    | `String`             | `currentColor` |
| xSmall   | `Boolean`            | ---            |
| small    | `Boolean`            | ---            |
| dense    | `Boolean`            | ---            |
| size     | `Boolean`            | `24`           |
| large    | `Boolean`            | ---            |
| xLarge   | `Boolean`            | ---            |
| disabled | `Boolean`            | ---            |
| rotate   | `String` or `Number` | ---            |
